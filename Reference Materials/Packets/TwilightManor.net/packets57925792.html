byte 0 Packet ID (0x98 - Character Allnames)
ushort 1 Blocksize
uint 3 Thing UID
char[30] 7 Thing Name (Included ONLY when sent by the server)
When the client sends this packet, it does not include the 'Thing Name,' and the 'Thing UID' is the UID of the thing which the client wants to know the name of. The server should respond with the server-version of this packet.

When the server sends this packet, it includes the 'Thing Name,' making the packet 37 bytes long.
