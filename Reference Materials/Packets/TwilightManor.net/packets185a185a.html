Sent By: Client0x07 - Item Pickup Request (7 bytes)
Variable TypeStart IndexPacket/Variable Name
byte 0 Packet ID (0x07 - Item Pickup Request)
uint 1 UID to pick up
ushort 5 Number of items in stack
The UID should always be checked to be sure it exists, is an item, and is grabbable.

This is sent when the client tries to drag an item. The client will send either packet 0x08
or 0x13 when it wants to put the item down somewhere.
