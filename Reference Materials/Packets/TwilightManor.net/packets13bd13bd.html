Sent By: Client0x13 - Item Equip Request (10 bytes)
Variable TypeStart IndexPacket/Variable Name
byte 0 Packet ID (0x13 - Item Equip Request)
uint 1 UID of the item being dragged
byte 5 Layer - Should be ignored. The server should determine the layer the item goes on (if it can even be equipped at all) itself.
uint 6 Target UID (The creature they're trying to drop it on)

The UID should be checked to make sure that it is already held (In the dragging layer).

This is sent when the client drops an item they've picked up onto themselves or another creature.

This packet will NOT be sent when dropping the item on the ground or into a container. Instead, 0x08 is sent.
