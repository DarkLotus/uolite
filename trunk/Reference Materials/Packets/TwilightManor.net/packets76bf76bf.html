Sent By: Server0xA8 - Server List (6+numServe bytes)
Variable TypeStart IndexPacket/Variable Name
byte 0 Packet ID (0xA8 - Server List)
ushort 1 blockSize
byte 3 Flag
ushort 4 Number of servers in the list
ushort 6+server*40 Server # (Should go from 0 to 1 less than the number of servers listed. UOX starts at 1 instead. The client doesn't care, but the byte positions on this page were written with the server# starting at 0. If you start at one, change +server*40 to +(server-1)*40)
char[30] 8+server*40 Server name
ushort 38+server*40 Unknown (Jerrith's says this is part of the server name, but I disagree. Of course, I could be wrong.)
byte 40+server*40 Percent full (0-100)
sbyte 41+server*40 Time zone the server is in
byte[4] 42+server*40 Server IP address (to ping). Has to be sent in reverse order. For example, 192.168.0.1 is sent as [01 00 A8 C0]
Jerrith's calls "Flag" "System Info Flag," and says 0xCC means don't send and 0x64 means send video card. RunUO and SteamEngine send 0x5d, Sphere and UOX send 0xFF.

*40 is necessary since each server needs 40 new bytes.

Client will respond with 0xA0(Server Select)
