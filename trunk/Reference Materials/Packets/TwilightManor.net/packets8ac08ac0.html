Sent By: Client0x08 - Item Drop Request (14 bytes)
Variable TypeStart IndexPacket/Variable Name
byte 0 Packet ID (0x08 - Item Drop Request)
uint 1 UID to put down
ushort 5 X coordinate
ushort 7 Y coordinate
sbyte 9 Z coordinate
uint 10 Container UID or 0xffffffff (meaning on the ground)
The UID should be checked to make sure that it is already held (In the dragging layer).

This is sent when the client tries to put down or place into a container the item which it was dragging.

If the container uid is 0xffffffff, then the item is being dropped on the ground at the specified x/y/z coordinates.

If the container UID is NOT 0xffffffff, then the item is being dropped in the container. The container UID should be checked to be sure it exists, and is actually a container.

This packet will NOT be sent when dropping the item on a creature. Instead, 0x13 is sent.
