Variable TypeStart IndexPacket/Variable Name
byte 0 Packet ID (0x1A - Item Details)
ushort 1 blockSize
uint 3 Item's UID - Includes 0x40000000. If the item has an amount > 1, also includes 0x80000000 (uid|0xc0000000).
ushort 7 Item dispid (model #) - If this includes 0x8000, 'Increment model #' is sent.
ushort 9 Amount, or corpse body dispid - Optional
byte 11 Increment model # by this amount, included if model # includes 0x8000. Useless?
ushort 12 X position of item. If this includes 0x8000, 'direction' is sent.
ushort 14 Y position of item. If this includes 0x8000, 'color' is sent. If this includes 0x4000, 'flags' is sent.
byte 15 Direction (Unused?)
sbyte 16 Z position of item
ushort 17 Color. Only sent if Y includes 0x8000.
byte 19 Flags. Only sent if Y includes 0x4000.
Byte positions shown for this packet assume all optional fields are used. You can figure out the real byte positions if particular fields aren't included. If no optional fields are sent, the packet will only be 14 bytes long. If ALL optional fields are sent, the packet will be 20 bytes long.
